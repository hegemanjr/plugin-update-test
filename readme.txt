Plugin Update Checker

== Description ==

This plugin modifies the admin bar menu provided by WP Multi Network by grouping sites by network.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress


== Frequently Asked Questions ==

= Does this work on single site installs? =

No. This is for a WordPress environment with multiple multisite networks using the WP Multi Network plugin

= Will this change the menus customers would use on my websites? =

No. This will only modify the menu in the admin bar, which you only see if you are a logged in user.
